Lien Github : https://github.com/GColloch/React-Node-To-do-List

Coté backend :

# Dépendance à installer :

npm : npm i
Mongoose : npm install mongoose
Express : npm install express

Variable d'environnement : 
MONGODB_URI=mongodb://localhost:27017/mytaskmanagerdb
PORT=5000

Démarrer le serveur : npm start



Coté frontend :

# Dépendance à installer :

yarn : yarn install
react-router-dom : yarn add react-router-dom ou npm install react-router-dom
Axios : yarn add axios
material-ui : yarn add @material-ui/core
scss : yarn add sass

Démarrer le client : yarn start
